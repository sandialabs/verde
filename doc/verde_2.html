Element Types

Verde will analyze the following element types:

Hexes Tetrahedra
Pyramids Wedges (triangular prisms)
Knives Quads
Triangles Bars (beams)

Verde will handle all higher-order variants of these types provided they are defined using the standard Exodus II node numbering; however, for higher-order elements, the metrics are calculated based on the linear versions of the elements.

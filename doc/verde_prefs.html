Perspective View

Perspective view can be toggle on or off. When off, viewing is orthographic (lines perpendicular to the screen do not appear to converge).

AutoFit

Verde's AutoFit option centers and zooms drawn entities after issuing a graphics command that causes the screen to redraw, as with:
  • activating or deactivating element blocks,
  • issuing a draw command (ie the command 'draw (hex|tet|wedge| pyramid|knife|quad|tri|edge|node) <range>') from the command window.

Quick Transform

This toggle controls whether mouse activated transformations (rotate, pan, or zoom) display the entire model or just the model edges. Transformations occur much more smoothly and quickly if only the model edges are displayed during the transformation process. The surface model can be displayed during transformations by toggling this option off.

Reset

This function returns the graphics model to its default rotation, scale, and translation, such that the model center is located at the center of the display region.
